Process

One endpoint. Four stages.

  1. Process the Excel workbook into structured JSON.
  2. Generate insights, trends, risks, and an executive summary with OpenAI.
  3. Create a professionally formatted PowerPoint deck from your reference file and workbook data.
  4. Return the finished `.pptx` file as a direct download.

Inputs

  • Excel workbook: `.xlsx`, `.xls`, `.xlsm`
  • Reference file: `.pptx`, `.potx`, `.pdf`
  • Output: downloadable PowerPoint report
